Project title and description: Office Hour Scheduling Assistant--Enables a more efficient way to allow students and teachers/professors to develop office hours that are more accessible to students availability.
Features implemented: Login page, student dashboard, and professor dashboard 
Technology stack used: React.js, typescript, and HTML.
Installation/setup instructions (step-by-step): Install Vite and run and local host 
How to run the application: Local Host using Vite.
